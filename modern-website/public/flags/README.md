# Country Flags Directory

This directory contains country flag images for the BeezeeProcessSection component.

## File Naming Convention

Flag images should follow this naming convention:
- `KE.png` - Kenya
- `NG.png` - Nigeria  
- `ZA.png` - South Africa
- `GH.png` - Ghana
- `UG.png` - Uganda
- `RW.png` - Rwanda
- `TZ.png` - Tanzania

## Image Specifications

- **Format**: PNG (recommended) or JPG
- **Size**: 128x80 pixels (optimal for display)
- **Aspect Ratio**: 16:10 (standard flag ratio)
- **Quality**: High quality, sharp edges
- **Background**: Transparent (PNG) or appropriate flag background

## How to Add Flags

1. Upload your flag images to this directory
2. Ensure they follow the naming convention above
3. The component will automatically use the images if they exist
4. If an image is missing or fails to load, it will fall back to emoji flags

## Current Status

The system is set up to use custom flag images. Once you upload the flag files, they will replace the emoji flags in the country cards.

## Fallback System

If a flag image is not found or fails to load, the component will automatically display the emoji flag instead, ensuring the display always works correctly.
